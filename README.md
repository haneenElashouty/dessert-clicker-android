# 🍰 Dessert Clicker

A modern Android application built to practice the **M-shaped professional model**, bridging the gap between high-fidelity UI design and robust software architecture.

## 🚀 The Mission
The goal was to move beyond static UI and implement a **Single Source of Truth** using Android's ViewModel and StateFlow, ensuring the app is both beautiful and logically sound.

## 🛠️ Tech Stack & Concepts
- **Jetpack Compose:** For a modern, declarative UI.
- **ViewModel & StateFlow:** To handle state management and business logic.
- **JUnit:** For unit testing core game mechanics.
- **Material 3:** For a polished look with rounded corners and centered layouts.

## 🧠 Key Learnings
- Separating UI (Stateless) from Logic (ViewModel).
- Handling data transitions through `copy()` to maintain immutability.
- Writing testable code from the ground up.
